
Machine Learning Internship Tasks

This repository contains Machine Learning internship projects implemented using Python, Scikit-Learn, OpenCV, NumPy, Pandas, and Matplotlib.

Task 01: House Price Prediction using Linear Regression
Developed a Linear Regression model to predict house prices based on square footage, number of bedrooms, and number of bathrooms.

Techniques Used

Linear Regression
Data Preprocessing
Train-Test Split
Model Evaluation

Task 02: Customer Segmentation using K-Means Clustering
Implemented a K-Means clustering algorithm to group retail store customers based on their purchase history.

Techniques Used

K-Means Clustering
Elbow Method
Customer Segmentation
Data Visualization
Dataset: Customer Segmentation Dataset (Kaggle)

Task 03: Cats vs Dogs Image Classification using SVM
Built an image classification model using Support Vector Machine (SVM) to classify images of cats and dogs.

Techniques Used

OpenCV
Image Preprocessing
Feature Extraction
SVM Classification
Dataset: Dogs vs Cats Dataset (Kaggle)

Task 04: Hand Gesture Recognition
Developed a hand gesture recognition system capable of identifying and classifying different hand gestures from image data.

Techniques Used

OpenCV
Image Processing
SVM Classification
Gesture Recognition
Dataset: LeapGestRecog Dataset (Kaggle)

Task 05: Food Recognition and Calorie Estimation
Developed a machine learning model to recognize food items from images and estimate their calorie content.

Techniques Used

Image Classification
OpenCV
SVM
Food Recognition
Dataset: Food-101 Dataset (Kaggle)

Technologies Used
Python
NumPy
Pandas
